# Ruboty::SyoboiCalendar
Ask today's Japanese anime line-up from [cal.syoboi.jp](http://cal.syoboi.jp/).

## Installation
```ruby
# Gemfile
gem "ruboty-syoboi_calendar"
```

## Usage
```
@ruboty list anime
```

## ENV
```
SYOBOI_CALENDAR_CHANNEL_IDS - Comma-separated channel IDs to filter programs
```
